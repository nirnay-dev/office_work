📄 PDF Batch Downloader

📖 How to Use:

  1. Login to the iGRASP Portal in a separate browser tab.
  2. Paste the URL of any project page in the box below.
  3. Set the Start/End IDs (e.g., 15 down to 0).
  4. Browser Setup: Go to Settings > Downloads and turn OFF "Ask where to save each file".
  5. Permissions: When you click Start, look at your URL bar and click "Allow Pop-ups" for this site.
  6. The script will visit each page, find the official Print link, and trigger a download.
